Sample Works

First Cisco Project (Fall 2012)
My mentor at Cisco Systems coordinated my team to create a device that would relay communication from the user to a Cisco proprietary set-top box. This would be used in the testing lab in order to ease the workers' job during testing. We used the Arduino micrcontroller along with the EasyVR module to create a voice recognizing device that would act as a remote control. We had commands such as 'channel up', 'channel down', 'menu', 'guide', and so on.

Second Cisco Project (Fall 2013 - Spring 2014)
Mentor coordinated my same team to create an Android application that would display set-top box diagnostic data for lab workers. This involved parsing data from a JSON file that would be generated after a set-top box would perform a system update or reboot. Data would be shown in tabular format detailing information such as the last reboot, last update, number of reboots, and so on. Additionally, the application doubled as a remote control, in order to easily display data while also testing components out.
